# App Store Connect Setup Guide - Step by Step

## Prerequisites

### Required Accounts
- **Apple Developer Account** ($99/year) - [Enroll here](https://developer.apple.com/programs/enroll/)
- **Apple ID** (free) - [Create here](https://appleid.apple.com)

## Step 1: Apple Developer Program Enrollment

### 1.1 Check if You Already Have an Account
1. Go to [Apple Developer](https://developer.apple.com)
2. Click "Account" in the top right
3. Sign in with your Apple ID
4. Check if you see "Apple Developer Program" membership

### 1.2 Enroll in Apple Developer Program (if needed)
1. Go to [Apple Developer Program](https://developer.apple.com/programs/enroll/)
2. Click "Enroll"
3. Sign in with your Apple ID
4. Choose your entity type:
   - **Individual**: You as a person
   - **Organization**: Company/team (requires D-U-N-S number)
5. Fill out required information:
   - Personal details
   - Contact information
   - Payment method ($99/year)
6. Submit for review (usually approved within 24-48 hours)

## Step 2: Access App Store Connect

### 2.1 Sign In to App Store Connect
1. Go to [App Store Connect](https://appstoreconnect.apple.com)
2. Sign in with the same Apple ID used for Developer Program
3. You should see the App Store Connect dashboard

### 2.2 First-Time Setup (if applicable)
- Complete any required agreements
- Set up your team information
- Configure payment and tax information

## Common Issues and Solutions

### Issue: "Bundle ID Already in Use"
**Solution**: 
- Choose a different bundle ID (e.g., com.infinitefootball.scanningpro.app)
- Update your Xcode project to match

### Issue: "Apple Developer Program Not Active"
**Solution**:
- Check your enrollment status
- Ensure payment is processed
- Contact Apple Developer Support if needed

### Issue: "Cannot Access App Store Connect"
**Solution**:
- Verify you're signed in with the correct Apple ID
- Ensure you have the correct role/permissions
- Check if your Developer Program membership is active
