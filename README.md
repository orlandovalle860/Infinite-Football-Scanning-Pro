# Infinite Football Website

A professional website for Infinite Football, showcasing the Infinite Football Scanning Pro app, professional training services, and branded merchandise.

## 🌟 Features

### Homepage (`index.html`)
- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **Features Section**: Highlights key benefits of Infinite Football
- **App Showcase**: Detailed overview of the scanning app
- **Training Services**: Overview of professional training offerings
- **Call-to-Action**: Encourages visitors to download app or book training

### App Page (`app.html`)
- **App Hero**: Detailed app introduction with badges
- **Features**: Comprehensive app feature breakdown
- **How It Works**: Step-by-step guide to using the app
- **Training Modes**: Detailed explanation of different scanning modes
- **Download Section**: App Store integration and download options

### Training Page (`training.html`)
- **Training Hero**: Professional training introduction
- **Services**: Detailed breakdown of training offerings
  - Individual Training Sessions ($75/session)
  - Group Training Programs ($25/player/session)
  - Coaching Certification Program ($299/complete program)
- **Process**: 4-step training methodology
- **Testimonials**: Customer success stories
- **Booking Form**: Integrated booking system

### Merchandise Page (`merchandise.html`)
- **Product Catalog**: 6 featured products with pricing
- **Collections**: Organized product categories
- **Shopping Cart**: Full e-commerce functionality
- **Product Modals**: Quick view product details
- **Responsive Design**: Mobile-friendly shopping experience

### Contact Page (`contact.html`)
- **Contact Information**: Multiple ways to get in touch
- **Contact Form**: Professional form with validation
- **FAQ Section**: Common questions and answers
- **Business Hours**: Clear availability information

## 🎨 Design Features

## 🚀 Getting Started

### Prerequisites
- A web server (local or hosted)
- Modern web browser
- Images for the website (see `images/placeholder.txt`)

### Installation
1. **Download/Clone** the website files to your server
2. **Add Images**: Place your images in the `images/` directory
3. **Customize Content**: Update text, prices, and contact information
4. **Test**: Open `index.html` in your browser

## 🔧 Customization

### Colors
The website uses a consistent color scheme defined in `css/style.css`:
- **Primary Blue**: `#007AFF` (App focus)
- **Success Green**: `#28a745` (Training focus)
- **Purple**: `#6f42c1` (Merchandise focus)
- **Orange**: `#fd7e14` (Contact focus)

### Content Updates
- **Text**: Edit HTML files to update content
- **Pricing**: Update prices in `training.html` and `merchandise.html`
- **Contact Info**: Update email and business hours in all pages
- **Images**: Replace placeholder images with your actual images

### Form Integration
The contact and booking forms are currently set up for demonstration. To make them functional:

1. **Contact Form**: Integrate with your email service or form handler
2. **Booking Form**: Connect to your booking system (Calendly, etc.)
3. **Shopping Cart**: Integrate with payment processor (Stripe, PayPal, etc.)
