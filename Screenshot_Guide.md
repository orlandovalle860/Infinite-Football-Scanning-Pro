# App Store Screenshot Guide - Infinite Football Scanning Pro

## Required Screenshots

You need screenshots for these device sizes:
- **6.7" iPhone** (iPhone 14 Pro Max, 15 Pro Max) - 1290 x 2796 pixels
- **6.5" iPhone** (iPhone 11 Pro Max, XS Max) - 1242 x 2688 pixels  
- **5.5" iPhone** (iPhone 8 Plus, 7 Plus) - 1242 x 2208 pixels
- **12.9" iPad Pro** (2nd generation and later) - 2048 x 2732 pixels

## How to Capture Screenshots

### Method 1: Using Xcode Simulator
1. Open your project in Xcode
2. Select the appropriate device simulator
3. Run the app (⌘ + R)
4. Navigate to the screen you want to capture
5. Take screenshot: ⌘ + S (or Device → Screenshot in menu)
6. Screenshots are saved to Desktop by default

### Method 2: Using Physical Device
1. Connect your iPhone/iPad to your Mac
2. Open Xcode and select your device
3. Run the app on the device
4. Navigate to the desired screen
5. Take screenshot on device: Power + Volume Up (or Power + Home on older devices)
6. Screenshots are saved to Photos app

## Screenshot Best Practices

### Visual Quality
- **Resolution**: Use the exact pixel dimensions listed above
- **Quality**: Ensure crisp, clear images
- **Brightness**: Since your app uses dark mode, ensure good contrast
- **No Status Bar**: Screenshots should not include the iOS status bar

### Content Guidelines
- **Show Real Features**: Don't use mockups - show actual app functionality
- **Highlight Key Features**: Focus on the most important aspects
- **Consistent Style**: Use similar visual style across all screenshots
- **Readable Text**: Ensure any text in the app is clearly readable

### Text Overlays (Optional)
You can add text overlays to highlight key features:
- Keep text short and impactful
- Use consistent font and styling
- Position text to not cover important UI elements
- Consider using your app's color scheme

## Upload to App Store Connect

### File Requirements
- **Format**: PNG or JPEG
- **Size**: Must match exact dimensions
- **File Size**: Under 10MB per screenshot
- **Naming**: Use descriptive names (e.g., "main-interface-6.7-inch.png")

### Upload Process
1. Go to App Store Connect
2. Select your app
3. Go to "App Store" tab
4. Scroll to "Screenshots" section
5. Drag and drop screenshots to appropriate device size
6. Arrange in desired order (first screenshot is most important)

## Testing Your Screenshots

### Before Uploading
- View screenshots on different devices/screens
- Ensure text is readable at small sizes
- Check that important features are visible
- Verify colors look good on different displays

### Common Issues to Avoid
- **Blurry Images**: Ensure high resolution
- **Cut-off Content**: Make sure important UI elements are fully visible
- **Poor Contrast**: Ensure text is readable against backgrounds
- **Inconsistent Style**: Keep visual style consistent across all screenshots
